Greenery version 1.0 11/03/2017

The Greenery app that encourages users to do phisical activity
more than using public transportation or cars to travel.

It shows the step you have walked and how many calories you have burnt.
Also, it calculates how much you save the environment by reducing
the usage of cars which emit carbon dioxide to harm the environment.
